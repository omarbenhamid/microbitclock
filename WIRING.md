Buzzer :
- Pin => Pin 0 (Green Wire)
- Pin => GND (Blue Wire)

OLED Screen :
- SDA Pin => 20 (Black Wire)
- SCL => 19 (Green Wire)
- GND => GND (Blue Wire)
- VCC => 3V (White Wire)

DS1302 RTC:
- VCC => 3V (White Wire)
- GND => GND (Blue Wire)
- CLK => 13 (Orange Wire)
- DAT => 14 (Black Wire)
- RST => 15 (Green Wire)
